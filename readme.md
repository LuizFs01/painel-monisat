# Painel

Repositório privado do projeto **Painel**.

Para clonar e enviar alterações, é necessário ter sido adicionado como **colaborador** e ter aceitado o convite no GitHub.

## Clonar o repositório

No terminal (PowerShell, CMD ou Terminal), execute:

```bash
git clone https://github.com/LuizHF01/painel.git
```

Entre na pasta do projeto:

```bash
cd painel
```

## Trabalhar no projeto

Faça as alterações necessárias no código ou crie novos arquivos normalmente.

## Subir alterações

Verifique os arquivos modificados:

```bash
git status
```

Adicione os arquivos:

```bash
git add .
```

Crie o commit:

```bash
git commit -m "Descrição da alteração"
```

Envie para o repositório:

```bash
git push
```

---

## Observações

- Apenas colaboradores com permissão podem enviar alterações
- Certifique-se de que o convite foi aceito no GitHub
- Cada colaborador utiliza sua própria conta GitHub
